# Plan
- Use up all the spare 2.5" hard drives
- Create a cold storage solution in addition to all pre-existing

# Constraints
- Absolute minimal spend
- In addition to no new storage (previous projects) no new compute
- Couldn't use preferred Unraid because no spare x86 compute

# Infrastructure

- Raspberry Pi 4 4GB
- mBeat 7 Port Powered USB Hub (4x USB3, 3x USB2)
- 6x Various 250GB-1TB 2.5" HDDs in external drive enclosures

# Steps
- Raspberry Pi Imager
  - Rpi OS Lite (no desktop)
- Update packages: `sudo apt update && sudo apt upgrade && sudo apt autoclean && sudo apt autoremove`
- Install relevant packages: `sudo apt install mergerfs snapraid -y`
- Identify relevant disks: `lsblk -o NAME,SIZE,TYPE,FSTYPE,UUID`
- Partition: `sudo parted /dev/sdX --script mklabel gpt mkpart primary ext4 0% 100%`
- Format :`sudo mkfs.ext4 -O ^has_journal /dev/sdX1`
